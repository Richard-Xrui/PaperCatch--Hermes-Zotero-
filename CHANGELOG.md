# PaperCatch 更新日志

## 2026-07-07 - 全面重构与中式美学升级

### 🎨 前端完全重构 - 中式美学设计

#### 设计理念
采用传统中式美学元素，打造简约高级的论文浏览体验：

- **水墨配色系统**
  - 宣纸白 (`#faf8f3`) - 主背景
  - 温润米白 (`#f5f2ea`) - 卡片背景
  - 浓墨 (`#2a2620`) - 主文字
  - 朱砂红 (`#c8312b`) - 印章与强调
  - 石绿 (`#3d5a45`) - 点缀色

- **宣纸质感**
  - 细微纸纹纹理背景
  - 柔和的阴影与描边
  - 温润的视觉体验

- **书法排版**
  - 使用中文衬线字体（思源宋体、宋体）
  - 优雅的字体层次
  - 合理的留白与间距
  - 字间距微调提升阅读体验

- **印章元素**
  - 品牌标识采用朱红印章风格
  - "纸" 字印章设计
  - "纸上得来" 品牌名 - 取自"纸上得来终觉浅"

#### 界面组件

1. **顶部导航栏**
   - 印章风格 Logo
   - 搜索框（竹简风格）
   - 最后更新时间
   - 功能按钮（刷新、设置、Hermes）

2. **左侧筛选器（卷轴风格）**
   - 日期范围快捷选择
   - 分类筛选（支持自定义）
   - 质量评分滑块
   - 排序方式
   - Zotero 状态筛选
   - 自动刷新选项

3. **论文卡片**
   - 宣纸质感背景
   - 清晰的层次结构
   - 标题、作者、摘要
   - 标签与徽章
   - 快捷操作按钮
   - 悬停效果

4. **模态框**
   - 详情查看
   - Hermes 智能搜索
   - 搜索设置

### 🎯 技术栈

- **前端**: 纯 HTML + CSS + JavaScript（零依赖）
- **后端**: Python + HTTP Server
- **数据**: JSON 文件存储
- **API**: RESTful 接口

- **桌面端** (>1024px): 左右分栏布局
- **平板端** (768px-1024px): 筛选器顶部折叠
- **移动端** (<640px): 单列布局，优化触摸操作

### 🎨 设计细节

1. **色彩层次**
   - 背景：宣纸白色系
   - 文字：墨色深浅层次
   - 强调：朱砂印章红
   - 点缀：青绿石色

2. **字体系统**
   - 正文：思源宋体/宋体（衬线）
   - UI：Inter（无衬线，fallback）
   - 字号：16px 基准
   - 行高：1.75（舒适阅读）

3. **留白设计**
   - 卡片间距：1.5rem
   - 内容边距：1.8rem
   - 组件间距：2rem

4. **交互反馈**
   - 悬停：轻微上浮 + 阴影加深
   - 点击：视觉反馈
   - 动画：0.2-0.3s 缓动

## 使用指南

### 启动服务

```bash
# 开发模式（自动重载）
python start.py

# 生产模式（无浏览器）
python start.py --no-browser

# 指定端口
python start.py --port 8888
```

### 访问地址

打开浏览器访问：http://localhost:8765

### 获取论文数据

```bash
# 获取最近 3 天的论文
python daily_pipeline.py --days 3 --max-per-cat 10

# 获取今天的论文
python daily_pipeline.py --days 0 --max-per-cat 25
```

### 配置 Zotero

```bash
# 交互式配置
python start.py --setup

# 或直接编辑 config.local.json
```
